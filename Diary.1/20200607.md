# R package
plyr: the split-apply-combine paradigm for R.
```r
a_ply(
  .data,
  .margins,
  .fun = NULL,
  ...,
  .expand = TRUE,
  .progress = "none",
  .inform = FALSE,
  .print = FALSE,
  .parallel = FALSE,
  .paropts = NULL
)
```
`.data`: bigwig.files
`.margins`: 1, split up `data` by. 1 splits up by rows
`.fun`: function to apply to each piece
