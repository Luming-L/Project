[bedtools intersect](https://bedtools.readthedocs.io/en/latest/content/tools/intersect.html)
```bash
bedtools intersect [OPTIONS] -a <FILE> \
                             -b <FILE1, FILE2, ..., FILEN>
```

`--filenames`
`-f 1.0`
